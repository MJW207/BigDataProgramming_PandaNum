## 3. 다운로드 명령 템플릿

```bash
# 시설 작물 받을 때
aihubshell -datasetkey 153 -filekey <파일키>

# 노지 작물 받을 때
aihubshell -datasetkey 147 -filekey <파일키>
```

## 4. 자동화 스크립트 예시 (시설 02.고추 통째 받기)

```bash
#!/bin/bash
# 시설 02.고추 전체 다운로드
KEYS_TRAIN=(46849 46850 46851 46852 46853)
KEYS_VAL=(46744 46745 46746)

for k in "${KEYS_TRAIN[@]}" "${KEYS_VAL[@]}"; do
    aihubshell -datasetkey 153 -filekey $k
done
```
